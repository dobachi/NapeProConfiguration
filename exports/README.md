# エクスポート済み設定 (バックアップ)

`tools/` のエクスポート機能で取得した Nape Pro 設定 JSON のバックアップ置き場。

- ファイル名は `nape-pro-settings-YYYY-MM-DD.json`（エクスポート日）。
- 復元は [`tools/`](../tools/) のインポート機能で行う。
- 公開リポジトリのため、コミット前に機密情報が含まれないことを確認すること
  （設定値・キーコードのみで、シリアル番号や個人情報は含まれない想定）。
